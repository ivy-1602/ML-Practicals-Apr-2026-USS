# 🤖 ML Diaries
_Eight algorithms. One dataset about Taylor Swift. Zero regrets._

## What's This?

My Semester 4 ML lab work — 8 Python programs that took me from _"what even is a feature matrix"_ to clustering Taylor Swift songs by vibe.

Each assignment is a case study. Each case study is a story. Some have Marvel heroes. One predicts eye color.

## Running the Code

Make sure you have the libraries:

```bash
pip install numpy pandas matplotlib scikit-learn
```

Then just:

```bash
python A1_supervised_unsupervised.py
```

Each file saves its own output graph as `A{n}_output.png` automatically.
No extra steps. No drama. (Unlike the actual ML concepts.)

## The Real Learning

**Supervised learning taught me:**
- That labels matter more than you think
- Accuracy isn't everything but it does feel good

**Unsupervised learning taught me:**
- Sometimes data just wants to find its own groups
- Centroids are the main characters nobody talks about

**Both taught me:** You can apply any algorithm to any dataset
if you're creative enough with the case study. Choose Taylor Swift. Always.
